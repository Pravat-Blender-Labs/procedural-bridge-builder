# 🌉 Procedural Bridge Builder

A powerful, mathematically precise Blender add-on that generates fully customizable, multi-material suspension bridges in real-time.

Whether you are building a cityscape or a sprawling canyon environment, this tool allows you to instantly generate complex architectural geometry with a single hotkey.

## ✨ Key Features
* **Real-Time Generation:** Tweak length, width, pillar count, and tower height instantly via the Redo Panel.
* **Mathematical Accuracy:** Uses pure Python and BMesh to calculate perfect parabolic suspension cables and dynamic pillar spacing.
* **Multi-Material Ready:** Automatically generates and assigns Eevee/Cycles-ready Node Materials (Asphalt, Painted Metal, and Steel).
* **Custom Color Picker:** Built-in UI to instantly repaint the towers and railings to match your scene.

## 📥 Installation
1. Go to the [Releases](../../releases) page and download the latest `Procedural_Bridge_v1.0.0.zip` file. **(Do not extract the .zip file!)**
2. Open Blender and go to **Edit > Preferences > Add-ons**.
3. Click the **Install...** button at the top right and select the downloaded `.zip` file.
4. Check the box next to **Add Mesh: Procedural Bridge Builder** to enable it.

## 🚀 How to Use
1. Open your 3D Viewport.
2. Press **`Ctrl + Shift + M`** to spawn the default bridge.
3. Look at the bottom-left of your screen for the **Adjust Last Operation** panel (or press **`F9`**).
4. Expand the panel to access the sliders. Drag the settings to watch your bridge update in real-time!
